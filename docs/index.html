Description

The ontology represents a shared research lab environment in which computers are registered with concrete technical properties such as storage capacity, read/write speed, RAM size and frequency, and CPU/GPU specifications. Each computer can be classified by access configuration (web, SSH, physical presence, AnyDesk) and by purpose (server or desktop). The model also supports software and hardware composition, as well as scheduling and user roles.
